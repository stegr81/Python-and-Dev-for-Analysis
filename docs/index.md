# Welcome to the Introduction to Python & Dev for Analysis course


## About the Course
This course is designed to ensure students leave with an awareness of some fundamental development skills and the knowledge to attempt a Python based approach to analysis and armed with modern development environment, set up and good to go!

The course is aimed at the absolute novice, <b>no prior experience is necessary</b> - we start at zero!

I endeavour to create an inclusive and relaxed learning environment where everyone is given room to learn and grow without undue pressure or expectation.

Briefly, we'll cover:

- Microsoft VS Code as your IDE (Integrated Development Environment); we'll get you set up and familiar
- Python fundamentals including strings, numbers, lists and more...
- An introduction to interacting with the command line in the terminal
- Version control with Git & GitHub
- Data ingestion and analysis in Python
- Getting AI assistance baked into VS Code. It is 2026 after all
